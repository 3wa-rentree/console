Chiffre 4

  1. Ouvrir un terminal
  2. Ecrire ls pour afficher les dossiers et fichiers dans le dossier actuel
  3. Ecrire cd texts /Aller dans le dossier "texts" pour
  4. cat t1.txt t2.txt | sort | head -n 1 | cut -c 1
  5. Le chiffre obtenu est le quatrième chiffre.
Commande Utilisation
ls Affiche la liste des fichiers dans un dossier.
cd Permet de changer de dossier (par exemple, aller dans un autre dossier).
cat Affiche le contenu d'un fichier dans le terminal.
sort Trie les lignes d'un fichier dans l'ordre alphabétique.
| Relie deux commandes, pour utiliser la sortie de l'une comme entrée de l'autre.
head Affiche les premières lignes d'un fichier.
cut Extrait une partie spécifique du contenu d'un fichier, comme une colonne ou des caractères.
