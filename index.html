Commande Utilisation Exemple
ls Affiche la liste des fichiers dans un dossier. ls
cd Permet de changer de dossier (par exemple, aller dans un autre dossier). cd texts
cat Affiche le contenu d'un fichier dans le terminal. cat t1.txt
sort Trie les lignes d'un fichier dans l'ordre alphabétique. sort t1.txt
| Relie deux commandes, pour utiliser la sortie de l'une comme entrée de l'autre. cat t1.txt | sort
head Affiche les premières lignes d'un fichier. L'option -n spécifie le nombre de lignes à afficher à partir du début du fichier. head -n 5 t1.txt
cut Extrait une partie spécifique du contenu d'un fichier, comme une colonne ou des caractères. L'option -c indique que l'on extrait des caractères. cut -c 1 t1.txt
